1. Set up the database:
   install MongoDB. 
   install npm.
2. Set up the server: 
   install Express.js at the package.json root. 
   install the Mongoose at the package.json root.
   install jade.
   install nodemon.
3. We connect database by Mongoose.
   Set namespace in Main.js.
   Check whether the route is working using Router.js.
   Pass the model and set url for collection using Collections.js.
   Check validation using Models.js.
   Show all things in Views.js.
   HTML page: index.html.
